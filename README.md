This repository contains the source code for **MedMinds**, an interactive informational website built as a high school project for the Romanian educational competition **Lider European**.

---

## Project Overview

**MedMinds** is a web platform focused on providing medical or health-related educational content in an accessible and user-friendly way. The site was designed and developed using core web technologies with emphasis on clarity, responsiveness, and ease of use.

The main goals of this project are:

- To provide reliable information on medical/health topics
- To present topics interactively and visually
- To create a web experience that is clean, responsive, and educational

---

## About the Competition ‚Äì *Lider European*

**Lider European** is a national Romanian educational initiative that emphasizes leadership, critical thinking, civic engagement, and project-based learning. It encourages students to develop solutions that address societal, scientific, or community topics.

This project was created specifically to participate in the *Lider European* competition, showcasing both technical skills and socially relevant content.
